# Varal das águas 

### Iniciar
Clone o repositório:

    git clone https://github.com/leosena777/desafio-inema-varal-das-aguas.git varaldasaguas

Abra a pasta criada:

    cd varaldasaguas

Inicie o projeto:

    npm start
    
o projeto deve iniciar automaticamente na porta 3000

### Sobre  o projeto

- Desenvolvido em React 
- Sass para estilos da pagina 
- Utiliza Bootstrap
